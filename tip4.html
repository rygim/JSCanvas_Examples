Animation

  • You may be able to get away with only drawing image deltas, but this becomes non trivial the more you add to your scenes
  • The canvas is not layered, it is essentially a buffered bitmap, so drawing something over something else will overwrite the other object
  • ctx.clearRect(x, y, width, height) is able to clear the canvas for you, but this will also take way the background

How do we do layers and perhaps only clear what we need to?

  • Canvases are transparent by default
  • Using some CSS, we can layer n number of canvases on top of one another.
  • This way, we can draw anything to separate layers and not worry about removing anything from a layer above or below the current canvas ctx
  • There is a little more overhead with n number of 2d canvas contexts
